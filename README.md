# Gameplay

You were **trapped** in the low tide. Your goal is to find the **3 magic objects** that help **raise the tide**.
However, these waters are far from calm; I hope your crew will be ready for any casualties.

## The rooms

<img width="771" height="307" alt="image" src="https://github.com/user-attachments/assets/664d24cf-000f-401a-beff-886d6de56004" />

### Motor

The engine is the most important room because it allows the submarine to move forward and backward. simple and necessary.

*Actions:*
* *Right button (hop in / leave)*
* *Left (speed backward) - Right (speed frontward)*

### Fins

The flippers room is also very important since it allows the submarine to change direction.

*Actions:*
* *Right button (hop in / leave)*
* *Left (rotate left) - Right (rotate right)*

### Cockpit

The cockpit allows you to probe the depths in order to steer the submarine through all the obstacles, but also... the dangers...
Objects to collect are in **YELLOW** and the exit is **RED**.

*Actions:*
* *Right button (hop in / leave)*
* *Left (rotate radar upward) - Right (rotate radar downward)*

### Electrical

As previously said, terrors lurk in these depths. Sometimes running is not enough anymore, you just have to turn off the lights and hope that it passes the way.

*Actions:*
* *Right Button (activate / deactivate)*

### Plank Storage

If you haven’t managed to be discreet, you may suffer some damage. Don’t panic! You just need to pick up a board and repair the potential holes. If you leave these holes for too long, you may never see the sun again...

*Actions:*
* *Right Button (pick up plank / give back plank)*

### Water Pump

Last resort, if you haven’t fixed the holes quickly enough, you will have to drain the water that has crept into your submarine. The pump is there for that! However, it doesn’t work miracles; you’ll need a lot of elbow grease if you want to survive.

*Actions:*
* *Right Button (pump water)*

# Start A Game

Once the game has been launched, you will need controllers (PS4/5, XBOX...). **Each connected controller is equivalent to one player**.

Once connected you can start right away !!
